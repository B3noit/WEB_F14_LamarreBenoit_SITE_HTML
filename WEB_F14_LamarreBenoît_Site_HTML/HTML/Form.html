- LES FORMULAIRES

-Qu'est-ce qu'un formulaire ?

Le formulaire est compris entre les balises FORM.
Un formulaire demande une multitude de données à l’utilisateur par le biais de différentes balises.
Les formulaires interactifs permettent aux auteurs de pages Web de doter leur page web d'éléments interactifs permettant par exemple un dialogue avec les internaute, à la manière des coupons-réponse présents dans certains magazines. Le lecteur saisit des informations en remplissant des champs ou en cliquant sur des boutons, puis appuie sur un bouton de soumission (submit) pour l'envoyer.

-La balise FORM

Les formulaires sont délimités par la balise <FORM> ... </FORM>, une balise qui permet de regrouper plusieurs éléments de formulaire (boutons,champs de saisie,...) et qui possède les attributs obligatoires suivants : METHOD indique sous quelle forme seront envoyées les réponses « POST » est la valeur qui correspond à un envoi de données stockées dans le corps de la requête, tandis que « GET » correspond à un envoi des données codées dans l'URL, et séparées de l'adresse du script par un point d'interrogation. ACTION indique l'adresse d'envoi et enfin, l'attribut ENCTYPE spécifie comment les données du formulaire sont traitées lorsqu'elles sont envoyées.
